# Painel Administrativo (SuperAdmin)

## O que é esta funcionalidade?

O Painel Administrativo é uma área restrita para administradores do sistema, permitindo gerenciar empresas, usuários, planos, assinaturas e configurações gerais do sistema.

## Como Acessar

- **Menu**: "Ferramentas" no menu principal (apenas para superadmin)
- **URL direta**: `/superadmin`

## Objetivo

- Gerenciar empresas cadastradas
- Gerenciar usuários do sistema
- Gerenciar planos e assinaturas
- Visualizar métricas do sistema
- Configurar ferramentas e assistentes
- Administrar o sistema como um todo

---

## Passo a Passo

### Acessar o Painel

1. **Faça login como superadmin**
   - Apenas usuários com perfil superadmin têm acesso

2. **Acesse o painel**
   - Clique em "Ferramentas" no menu
   - Ou acesse `/superadmin`

3. **Visualize o dashboard administrativo**
   - Métricas gerais do sistema
   - Resumo de empresas, usuários, etc.

### Gerenciar Empresas

1. **Acesse "Empresas"**
   - No menu do painel administrativo

2. **Visualize a lista de empresas**
   - Todas as empresas cadastradas

3. **Edite uma empresa**
   - Clique em "Editar"
   - Modifique dados da empresa
   - Atualize plano, status, vencimento, etc.

4. **Visualize detalhes**
   - Veja informações completas
   - Histórico, assinatura, etc.

### Gerenciar Planos

1. **Acesse "Planos"**
   - No menu do painel

2. **Visualize planos disponíveis**
   - Todos os planos cadastrados

3. **Crie um novo plano**
   - Defina nome, preço, funcionalidades
   - Configure limites e recursos

4. **Edite planos existentes**
   - Modifique características
   - Atualize preços e recursos

### Gerenciar Usuários

1. **Acesse a gestão de usuários** (se disponível)
   - Visualize todos os usuários

2. **Gerencie acessos**
   - Ative/desative usuários
   - Modifique permissões
   - Visualize atividades

### Visualizar Métricas

1. **No dashboard administrativo**
   - Veja métricas gerais
   - Empresas ativas, receita, etc.

2. **Analise dados**
   - Use para tomada de decisão
   - Identifique tendências

## Dicas e Observações

### Acesso Restrito:
- Apenas superadmins têm acesso
- Use com cuidado
- Alterações afetam todo o sistema

### Gerenciamento:
- Mantenha dados atualizados
- Monitore métricas regularmente
- Gerencie planos e assinaturas

## Perguntas Frequentes

**P: Quem pode acessar o painel administrativo?**
R: Apenas usuários com perfil superadmin.

**P: Posso criar novos planos?**
R: Sim, se você tiver acesso de superadmin.

**P: Como altero o plano de uma empresa?**
R: Acesse a empresa no painel administrativo e edite o plano associado.

---

**Nota**: Esta funcionalidade é apenas para administradores do sistema. Usuários normais não têm acesso.
