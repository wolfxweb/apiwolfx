# Manual do Usuário - Sistema CELX

## Índice Geral de Funcionalidades

Bem-vindo ao manual do sistema CELX! Este documento lista todas as funcionalidades disponíveis no sistema, organizadas por módulos. Cada funcionalidade possui um manual detalhado explicando como utilizá-la.

---

## 📋 Módulo: Autenticação e Perfil

### 1. Login e Registro
- **Arquivo**: `01_Login_e_Registro.md`
- **Descrição**: Como fazer login no sistema, criar uma nova conta e recuperar senha
- **Acesso**: Menu superior direito → "Entrar" ou "Cadastrar"

### 2. Perfil do Usuário
- **Arquivo**: `02_Perfil_do_Usuario.md`
- **Descrição**: Como visualizar e editar suas informações pessoais, alterar senha e gerenciar dados da empresa
- **Acesso**: Menu superior direito → Seu nome → "Perfil"

### 3. Dashboard Principal
- **Arquivo**: `03_Dashboard_Principal.md`
- **Descrição**: Visão geral do sistema com resumo de atividades e métricas principais
- **Acesso**: Menu principal → "Dashboard"

### 4. Planos e Assinaturas
- **Arquivo**: `04_Planos_e_Assinaturas.md`
- **Descrição**: Como visualizar planos disponíveis, assinar um plano e gerenciar sua assinatura
- **Acesso**: Menu superior direito → "Planos" ou através do perfil

### 5. Histórico de Pagamentos
- **Arquivo**: `05_Histórico_de_Pagamentos.md`
- **Descrição**: Como visualizar histórico de pagamentos e faturas
- **Acesso**: Menu superior direito → "Pagamentos"
