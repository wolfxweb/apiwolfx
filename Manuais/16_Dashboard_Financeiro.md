# Dashboard Financeiro

## O que é esta funcionalidade?

O Dashboard Financeiro fornece uma visão geral das finanças da empresa, incluindo receitas, despesas, saldo, contas a pagar e receber, e outros indicadores financeiros importantes.

## Como Acessar

- **Menu**: "Financeiro" → "Dashboard"
- **URL direta**: `/financial/dashboard`

## Objetivo

- Visualizar resumo financeiro da empresa
- Acompanhar receitas e despesas
- Monitorar contas a pagar e receber
- Analisar indicadores financeiros
- Ter acesso rápido a informações financeiras

---

## Passo a Passo

### Visualizar o Dashboard

1. **Acesse o Dashboard Financeiro**
   - No menu, clique em "Financeiro" → "Dashboard"

2. **Explore as seções disponíveis**
   - O dashboard pode conter diferentes widgets e gráficos
   - Cada seção mostra informações financeiras específicas

### Entender as Informações Exibidas

O dashboard pode mostrar:

- **Resumo Financeiro**
  - Saldo atual
  - Receitas do período
  - Despesas do período
  - Resultado (lucro/prejuízo)

- **Contas a Receber**
  - Total a receber
  - Vencidas
  - A vencer
  - Recebidas no período

- **Contas a Pagar**
  - Total a pagar
  - Vencidas
  - A vencer
  - Pagas no período

- **Fluxo de Caixa**
  - Entradas
  - Saídas
  - Saldo projetado
  - Gráfico de evolução

- **Indicadores**
  - Ticket médio
  - Margem de lucro
  - Outros KPIs relevantes

### Filtrar por Período

1. **Localize os filtros de data**
   - Geralmente no topo do dashboard

2. **Selecione o período desejado**
   - Hoje, Esta semana, Este mês, Este ano, Personalizado

3. **Visualize os dados atualizados**
   - Gráficos e métricas serão atualizados

### Navegar para Funcionalidades Específicas

1. **Use os cards/widgets**
   - Muitos elementos são clicáveis
   - Clique para ir à funcionalidade relacionada

2. **Use o menu Financeiro**
   - Acesse Contas a Receber, Contas a Pagar, Fluxo de Caixa, etc.
