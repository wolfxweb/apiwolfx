# Mais Vendidos no Mercado Livre

## O que é esta funcionalidade?

A funcionalidade de Mais Vendidos permite visualizar produtos mais vendidos por categoria no Mercado Livre, ajudando a identificar tendências e oportunidades de negócio.

## Como Acessar

- **Menu**: "Mercado Livre" → "Mais Vendidos"
- **URL direta**: `/ml/highlights`

## Objetivo

- Visualizar produtos mais vendidos por categoria
- Identificar tendências de mercado
- Encontrar oportunidades de negócio
- Analisar performance de categorias
- Comparar produtos por atributos

---

## Passo a Passo

### Visualizar Mais Vendidos

1. **Acesse a página de Mais Vendidos**
   - No menu, clique em "Mercado Livre" → "Mais Vendidos"

2. **Selecione uma categoria**
   - Use o seletor de categoria no topo
   - Escolha a categoria desejada

3. **Visualize a lista de produtos**
   - Produtos são ordenados por volume de vendas
   - Cada produto mostra informações relevantes

### Filtrar por Atributos

1. **Use os filtros disponíveis**
   - **Atributo**: Selecione um atributo específico (ex: Marca)
   - **Valor do Atributo**: Selecione o valor (ex: ID da marca)

2. **Aplique os filtros**
   - A lista será atualizada com produtos filtrados

3. **Remova os filtros**
   - Clique em "Limpar" para remover filtros

### Analisar Produtos

1. **Na lista de produtos**
   - Visualize informações de cada produto

2. **Compare produtos**
   - Veja preços, características, performance
   - Identifique padrões e tendências

3. **Use as informações para decisões**
   - Identifique produtos para adicionar ao catálogo
   - Analise preços de mercado
   - Entenda características dos produtos mais vendidos

### Exportar Dados

1. **Use a opção de exportação** (se disponível)
   - Pode estar em um botão ou menu

2. **Selecione o formato**
   - Excel, CSV, etc.

3. **Baixe o arquivo**
   - Os dados serão exportados para análise

## Recursos Disponíveis

### Informações Exibidas:

- **Dados dos Produtos**
  - Título do produto
  - Preço
  - Características principais
  - Atributos (marca, modelo, etc.)

- **Métricas**
  - Volume de vendas
  - Posição no ranking
  - Outras métricas relevantes

### Ações Disponíveis:

- Visualizar mais vendidos por categoria
- Filtrar por atributos
- Analisar produtos
- Exportar dados (se disponível)

## Perguntas Frequentes

**P: Com que frequência os dados são atualizados?**
R: Depende da sincronização. Os dados são atualizados quando você acessa ou sincroniza.

**P: Posso ver mais vendidos de todas as categorias?**
R: Geralmente é necessário selecionar uma categoria por vez para análise mais focada.

**P: Como uso essas informações para meu negócio?**
R: Use para identificar produtos para adicionar ao catálogo, analisar preços e entender tendências.

**P: Posso exportar os dados?**
R: Depende da funcionalidade disponível. Verifique se há opção de exportação.

**P: Os dados são específicos da minha conta?**
R: Não, os dados mostram produtos mais vendidos no Mercado Livre em geral, não apenas seus produtos.
