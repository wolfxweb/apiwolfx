# Dashboard Mercado Livre

## O que é esta funcionalidade?

O Dashboard do Mercado Livre fornece uma visão geral das métricas, análises e indicadores de performance relacionados às suas vendas e anúncios no Mercado Livre.

## Como Acessar

- **Menu**: "Mercado Livre" → "Dashboard"
- **URL direta**: `/analytics`

## Objetivo

- Visualizar métricas gerais de vendas no Mercado Livre
- Acompanhar performance de anúncios e campanhas
- Analisar indicadores de negócio
- Ter acesso rápido a informações importantes

---

## Passo a Passo

### Visualizar o Dashboard

1. **Acesse o Dashboard do Mercado Livre**
   - No menu, clique em "Mercado Livre" → "Dashboard"

2. **Explore as seções disponíveis**
   - O dashboard pode conter diferentes widgets e gráficos
   - Cada seção mostra informações específicas do Mercado Livre

### Entender as Informações Exibidas

O dashboard pode mostrar:

- **Métricas de Vendas**
  - Total de vendas no período
  - Número de pedidos recebidos
  - Receita total
  - Comparativo com período anterior

- **Performance de Anúncios**
  - Anúncios ativos
  - Anúncios pausados
  - Visualizações
  - Taxa de conversão

- **Métricas de Publicidade**
  - Campanhas ativas
  - Investimento em publicidade
  - Retorno sobre investimento (ROAS)
  - Produtos em campanhas

- **Indicadores de Negócio**
  - Ticket médio
  - Produtos mais vendidos
  - Categorias com melhor performance
  - Taxa de cancelamento

### Filtrar por Período

1. **Localize os filtros de data**
   - Geralmente no topo do dashboard
   - Pode haver opções como "Hoje", "Esta semana", "Este mês", "Personalizado"

2. **Selecione o período desejado**
   - Escolha um período pré-definido
   - Ou selecione datas personalizadas

3. **Visualize os dados atualizados**
   - Os gráficos e métricas serão atualizados conforme o período selecionado

### Filtrar por Conta do Mercado Livre

1. **Se você tiver múltiplas contas**
   - Use o filtro de conta no topo
   - Selecione a conta desejada
   - Ou selecione "Todas" para ver consolidadas

2. **Visualize os dados filtrados**
   - As métricas serão atualizadas para a conta selecionada

### Navegar para Funcionalidades Específicas

1. **Use os cards/widgets**
   - Muitos elementos do dashboard são clicáveis
   - Clique em um card para ir à funcionalidade relacionada

2. **Use o menu do Mercado Livre**
   - Acesse Anúncios, Pedidos, Publicidade, etc.
   - Cada módulo tem informações mais detalhadas

## Dicas e Observações

### Múltiplas Contas:
- Se você tem várias contas do Mercado Livre, pode filtrar por conta
- Ou visualizar dados consolidados de todas as contas
- Use o filtro para análises específicas

### Atualização de Dados:
- Os dados são atualizados automaticamente
- Algumas atualizações podem levar alguns minutos
- Use o botão de atualizar/refresh se necessário

### Interpretação de Métricas:
- Compare períodos para entender tendências
- Use os filtros para análises específicas
- Consulte os módulos detalhados para informações completas

### Navegação:
- Use os cards clicáveis para acesso rápido
- O menu sempre está disponível no topo
- Explore os módulos específicos para mais detalhes

## Perguntas Frequentes

**P: Por que não vejo todas as métricas no dashboard?**
R: Algumas métricas podem estar disponíveis apenas em planos específicos ou podem precisar de sincronização.

**P: Como atualizo os dados do dashboard?**
R: Os dados são atualizados automaticamente. Se necessário, recarregue a página ou use o botão de atualizar.

**P: Posso ver dados de todas as minhas contas do ML juntas?**
R: Sim, selecione "Todas" no filtro de contas para ver dados consolidados.

**P: Os dados do dashboard são em tempo real?**
R: Os dados são atualizados regularmente, mas podem ter um pequeno atraso dependendo da origem.

**P: Como acesso funcionalidades específicas a partir do dashboard?**
R: Clique nos cards/widgets ou use o menu "Mercado Livre" para navegar.

**P: Posso exportar os dados do dashboard?**
R: Depende da funcionalidade. Algumas seções podem ter opções de exportação.
